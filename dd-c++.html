C++ Assignment 1: Tic Tac Toe

This course taught me programming concepts for C++ from variables, loops, methods, inputs and outputs, drawing an image in the console using word characters, Object-oriented Programming, and AI.

The first assignment I did was a Tic Tac toe console application. It drew up the board using numbers 1 - 9 and x and o letters for x and o. The plan for this assignment was to make AI unbeatable, and code organized in classes dedicated functionality of the program.

It took a week to complete, but AI wasn’t unbeatable. It can be beaten. But even so, I did understand the concept of the assignment and got 70 in the end.

Here is the program running. You are the “X”, and the AI is “O”.
