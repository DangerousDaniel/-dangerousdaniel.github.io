Vex Monsters

Vex Monsters is a 2D fantasy platform game, where you save monsters from the evil betrayed.

This is the map, where the game take place.

It started back in December 2019 with my best friend and me, where dig into planning out the story, planing project and anything we needed to get it ready as a project for Nova Scotia Community College Capstone Program (Capstone, let you create a project with the help from industry).

The new year started, and we went off to find team members for our project. After a long road, we found a programmer and a person with a trader of Information Technology. We proposed to industry via network event at a bar in Halifax. By doing so, we found the project manager form immediaC, and he became our industry adviser.

From there, we started to plan enough to get started on the project from created a team charter to a game design document.

Then we started developing the game.

I am the Team Leader. I direct the team via Trello project board and weekly Scrums through Discord. I also program the logic and User Interface of the game from moving the player, ai, dialogue, NPC and other game mechanics. There even one more thing I do is writing the story from the overview, backstory, chapters, and any plan will need for the story.

The game is still under development stage and right now is nowhere near ready for the world.
